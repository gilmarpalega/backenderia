# Backenderia

Backenderia is a CMS API with common apps.

## Technology 

- Go
- Negroni
- gorilla/mux
- Angular.js
- Google Material Design Lite
- Caddy
- Docker / Docker-Compose

## Apps

- Pages
- News

## Run

- git clone https://github.com/guilhermebr/backenderia.git
- cd backenderia
- docker compose up -d
